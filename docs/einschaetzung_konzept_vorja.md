## Technischer Hinweis (nicht verhandelbar)

Dein Wunsch: *„Barren berührt Handy → PWA öffnet sofort im Vollbild-Fokus-Modus."*

Das funktioniert auf **iPhone nicht**. Apple sperrt Web NFC für Browser und PWAs vollständig. Es wird immer ein manueller Tipp auf einen System-Banner nötig sein – das ist eine Apple-Entscheidung, keine meiner Einschränkungen. Das muss vor der Hardware-Produktion klar sein. Warum überhaupt NFC? Warum nicht wenn schon so grandiose Holzmaterialen dann auch gleich mit Audio-Player? Umwege über Web liefern immer! Ausfallrisiken, Sicherheitsbedenken, ungewohnte Haptik und ne Menge anderer Dinge die dem Produkt nicht den Mehrwert geben der hier gefordert ist, egal wie toll so eine App ausschaut..
